# Project Structure

Projects MUST follow a consistent high‑level layout to simplify onboarding.

```text
MyProject/
├── src/
│   └── MyModule/
├── scripts/
├── docs/
└── tests/
```

- `src/` – modules and reusable code.
- `scripts/` – entrypoint scripts.
- `docs/` – documentation.
- `tests/` – automated tests.
